Type de spécialisation des menus

Valeurs énumérées:
NOLIST  Le menu n'est pas spécialisé
PLAYERLIST  Le menu est une liste de joueurs
PLAYERLIST_SANS_SPEC  Le menu est une liste de joueurs non spectateurs
CFGLIST  Le menu est une liste de configurations (voir DOSSIER_CONFIGURATIONS_MATCH)
MAPLIST  Le menu est une liste de lignes contenues dans un fichier (voir FICHIER_MAPLIST)
